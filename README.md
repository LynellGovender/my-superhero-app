 My Superhero App

A simple web application that showcases different Marvel superheroes, including their details and abilities.

## Features
- View different superheroes
- Click on a character to see more details
- Displays name, powers, and background info

## Technologies Used
- HTML
- CSS

## How to Use
1. Open the homepage
2. Click on any superhero image
3. View detailed information about the character
